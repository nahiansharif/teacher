AWS RDS integration


Thanks to the integration with AWS RDS, all of the data is stored in a centralized location, making it easy to extract and analyze data using a variety of tools, including MySQL Workbench and this website.






Search Bar


Users can look for a specific person in the database by typing their name. Search bar is going to show the specific details about the searched person based on the user's role.






Users


Each user has a unique view and permissions to use the database. Teachers can only update and view students’ grades. While admin can add, view, edit, and delete teachers and students. Also admin can create new username and password when they add new teachers. However, admins can't update students’ grades because that’s teachers’ job.






Stats


Admins can see the income, cost, number of teachers and students, and the profit after successful log in.
