Log In


A log in system for employees and managers authentications. The program takes the user’s input and search for username and password in database. If the username and password matches, the log in was a success. Otherwise, the program sends a warning message and give users another tries.






Employee


Employee can create order, they can add the requested to the cart, make necessary changes, proceed with payment, and print, email, or cancel receipt. The order is in pending list, and employee can view, cancel, or complete this pending order.






Manager


Managers can change the status of a non-pending order. They can view all the canceled and completed orders. IF an item is out of stock from the menu, manager can disable that item. Managers are the only one to close the application.






Search Bar


This search bar is going to assist users find specific receipt. Each Receipt has unique 7-digit numbers. Users have to type that 7-digit number to view all details of that receipt in the system. Employee can only view the receipt, but managers can change the status of the receipt.







Save and Retrieve Data


All of the receipt and its details are saved in computer. The data can be extracted, transformed and loaded. The receipt will always be going to keep the original data.
